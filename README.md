# StreaMRAKv1
Streaming, multi-resolution, adaptive, kernel solver

Cite the code: [![DOI](https://zenodo.org/badge/383231702.svg)](https://zenodo.org/badge/latestdoi/383231702)

https://github.com/LCSL/FALKON_paper.git
